\doxysubsection{Detailed Description}
An abstract class to evaluate function and gradients. 

@description Cpp\+AD has a very simple (and ugly) example for an interface to Ipopt. I decided to make an abstract class so that the functor can be set once the tree is built so that the objective function and the variables vector (which depends on the depth, number of feats and number of labels) can be passed later. \begin{DoxyNote}{Note}
